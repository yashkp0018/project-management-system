# Project Management System

## Overview
A web-based Project Management System developed using JavaScript, Node.js, Express.js, and MySQL. The application helps teams manage projects, tasks, users, and workflows efficiently.

## Features
- User Authentication
- Project Creation and Management
- Task Assignment and Tracking
- Role-Based Access Control
- Database Integration with MySQL
- REST API Architecture

## Technologies Used
- JavaScript
- Node.js
- Express.js
- MySQL

npm install
npm start
